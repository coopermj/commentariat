# Recommended Sources

This project targets public domain or freely redistributable commentaries.
Below are common sources to prioritize when collecting data.

## SWORD modules (public domain)
- Matthew Henry (Concise Commentary)
- Jamieson-Fausset-Brown (JFB)
- Albert Barnes' Notes
- John Gill's Exposition
- Adam Clarke's Commentary
- Keil & Delitzsch (OT)
- Geneva Notes (where licensing permits)
- Pulpit Commentary

## Other public domain sources
- CCEL scans (public domain editions)
- Archive.org public domain editions
- Wikisource public domain commentaries

For each source, record license metadata and attribution in the manifest.
